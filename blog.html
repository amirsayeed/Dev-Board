Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM. Some of the most common ways are :

  • getElementById() : It is used to select a specific element by assigning a unique id to it.
  • getElementsByClassName() : It is used to select elements with the same class name.
  • getElementsByTagName() : It is used to select elements by their tag name.
  • querySelector() : It selects only the first element that matches the specified css selector as it can take element by id, tag or class but always return the first occurence in the DOM.
  • querySelectorAll() : It returns all elements that match a css selector such as querySelectorAll(".btn") will return all the elements with class name as btn.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML gets or sets HTML material, including tags and structure which is suitable for dynamic HTML editing. InnerText deals with visible text content and ignore HTML elements and CSS styling which is appropriate for circumstances that require displayed text to be set or to get of an element. TextContent returns raw text content that includes whitespace and line breaks, as well as hidden text and it's value is not parsed as HTML.
