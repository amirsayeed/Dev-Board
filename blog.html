Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM. Some of the most common ways are :

  • getElementById() : It is used to select a specific element by assigning a unique id to it.
  • getElementsByClassName() : It is used to select elements with the same class name.
  • getElementsByTagName() : It is used to select elements by their tag name.
  • querySelector() : It selects only the first element that matches the specified css selector as it can take element by id, tag or class but always return the first occurence in the DOM.
  • querySelectorAll() : It returns all elements that match a css selector such as querySelectorAll(".btn") will return all the elements with class name as btn.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML gets or sets HTML material, including tags and structure which is suitable for dynamic HTML editing. InnerText deals with visible text content and ignore HTML elements and CSS styling which is appropriate for circumstances that require displayed text to be set or to get of an element. TextContent returns raw text content that includes whitespace and line breaks, as well as hidden text and it's value is not parsed as HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique which allows the parent element to handle events instead of attaching event listeners to child element individually. When an event occurs in a child element it will reach through the parent element using event bubbling concept where the event listener is attached to and by the use of event.target the element is identified inside the event listener and further operation are performed.

Question-4: What is event bubbling in the DOM?

When an event is attached to or performed on an element, the event is bubbled up or propagated to its parent and grandparent elements while triggering each ancestor elements event listeners until it reaches the root element.

If there is a div with class parent-div with a div with class child-div and that child-div has a button element where click event is performed and there are event listeners attched to parent-div, child-div and button element where it will console such element is clicked then the sequence will be:

Button is clicked

Child-div is clicked

Parent-div is clicked

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, document.createElement(element) is used, to add content to an element innerHTML, innerText or textContent are used and to remove an element remove() or its child removeChild() is used. For example:


1. Create an Element

const div = document.getElementById("para-container")

const p = document.createElement("p")

2. Add the element

p.innerText = "Hello Developer"

div.appendChild(p)

3. Remove the element

p.remove()
